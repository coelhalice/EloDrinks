## Elo Drinks

### Technologies

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Lupi](https://www.npmjs.com/package/lupi)

### Installation

```bash
$ npm install
```

### Running the app

- Set up the file .env based on the .env.example file.

```bash
# development
$ npm run dev
```

### Lint and Build

```bash
# Check lint and format the code
$ npm run lint

# Build the app
$ npm run build
```
